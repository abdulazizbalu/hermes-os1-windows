# Nur

**Nur** — локальный AI-помощник для офисных задач на Windows. Работает на Gemma 4 через Ollama, без API-ключей и без облака. Интерфейс на русском, с быстрыми задачами для русско- и узбекоязычных пользователей.

## Что умеет

### AI-задачи (на базе локальной Gemma 4)
- **Чат** — свободный диалог с моделью, разговоры сохраняются автоматически
- **Письма** — черновики писем по теме, тону, получателю и языку (RU/UZ/EN)
- **Суммировать** — краткое содержание длинных текстов с выбором длины и формата
- **Перевести** — RU ↔ UZ (латиница и кириллица) ↔ EN
- **Улучшить** — 7 режимов: грамматика, короче, длиннее, формальнее, дружелюбнее, проще, переписать

### Локальные утилиты (без AI и без интернета)
- **Конвертер** — единицы (длина/вес/объём/площадь), температура, валюты (USD/EUR/RUB/UZS/KZT/GBP/TRY/CNY), размер файла, число прописью (рус. + узб.)
- **Транслитерация** — RU кириллица → латиница, UZ кириллица ↔ латиница (с поддержкой oʻ/gʻ)

### Системные
- **История** — все разговоры с поиском, сохраняются в `%AppData%/Nur/history.json`
- **Настройки** — управление моделью Gemma (E2B / E4B / 26B), запуск Ollama

## Стек

- Electron 39 + React 19 + TypeScript 5.9
- Ollama (локальный движок) + Gemma 4 (E2B / E4B / 26B)
- Vite 7, Vitest, electron-builder NSIS

## Разработка

```sh
npm install
npm run dev
```

```sh
npm run typecheck   # TypeScript
npm run test        # Vitest (38 тестов)
npm run build       # сборка
```

## Сборка Windows-установщика

```sh
npm run build:win
```

Установщик появится в `release/` как `Nur-<version>-x64.exe`.

## Первый запуск

1. Запустите Nur
2. Перейдите в **Настройки**
3. Установите Ollama (кнопка «Скачать Ollama») если нужно
4. Запустите Ollama, нажмите «Подготовить Gemma» — модель скачается локально (≈3-9 ГБ)
5. Проверка русского — Nur ответит подтверждением
6. Вернитесь в **Чат** — готово к работе

## Архитектура

- `src/main/` — Electron main process: IPC handlers, Ollama HTTP client, file storage
- `src/preload/` — мост `window.os1` с типизированным API
- `src/renderer/` — React UI
  - `components/` — переиспользуемые (Sidebar, BootScreen, AiOutputPanel, etc)
  - `views/` — экраны (ChatView, EmailView, SummarizeView, ...)
  - `hooks/` — `useGemma` для AI-задач
  - `utils/` — конвертеры, транслитерация, число прописью
- `src/shared/` — типы и валидация для IPC payload
